Raw data: http://traces.cs.umass.edu/index.php/Smart/Smart

The raw data are stored in the data_raw folder.

Data preprocessing: data_processing.py

The cleaned data are stored in the data_cleaned folder.

Implementation: Chicago_point.py, Chicago_prob.py, pf.py, util.py
Denver_point.py is for point forecasting, and Denver_prob.py is for probabilistic forecasting. 

The results are stored in the results folder.
