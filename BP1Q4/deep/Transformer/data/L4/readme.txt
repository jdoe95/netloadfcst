Raw data: https://www.nyiso.com/load-data
PVWatts: https://pvwatts.nrel.gov/pvwatts.php

The raw data are stored in the data_raw folder.

Data preprocessing: None

The cleaned data are stored in the data_cleaned folder.

Implementation: point.py, prob.py, util.py, pf.py
point.py is for point forecasting, and prob.py is for probabilistic forecasting.

The results are stored in the results folder.
